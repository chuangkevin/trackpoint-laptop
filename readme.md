# Trackpoint Laptop 專案

> 最後更新：2026-02-26

## 專案介紹

以二手筆電主機板為基礎，結合長條螢幕與小紅點鍵盤，改裝成一台沒有觸控板的輕便筆電。

### 核心需求

- 小紅點鍵盤
- 長條螢幕
- Type-C 供電

---

## 硬體規格

- **規格**: i5-8250U / 8GB RAM / 256GB SSD
- **I/O**:
